## How to run locally
1. Clone the repository  
   `git clone https://github.com/dipak1221/rise_india_foundation_drafted_website.git`
2. Change into the project directory  
   `cd rise_india_foundation_drafted_website`
3. Open index.html in your browser (double-click) or run a local server:  
   - Install Live Server: `npm install -g live-server`  
   - Run: `live-server`
4. Use browser DevTools to inspect and tweak styles. Keep images in an assets/images or images folder and reference with relative paths.

## Deployment (GitHub Pages)
1. Go to the repository Settings → Pages.
2. Select the branch to publish (e.g., main or master) and root (/) folder.
3. Save — GitHub Pages will provide a URL to the published site.

Alternative: publish using a gh-pages branch or set up a GitHub Actions deployment.
